Se creará una interfaz amigable para el usuario. Tendremos una sección que podrá registrar una apromixación hacia el estado de ánimo del usuario en base a ciertas preguntas. Por otro lado tendremos la sección de eventos donde se podrán encontrar todos los eventos disponibles con fechas, una breve descripción, imágenes y costos. Adicional tendremos una búsqueda de eventos que te ayudará a filtrar y encontrar el evento perfecto. Complementaremos la aplicación con galerías de imagenes, una página de contacto y la posibilidad de registrar eventos como usuario. Nuestra aplicación se puede utilizar como usuario registrado o no registrado.

Definiremos los roles necesarios para nuestra aplicación. Un usuario registrado, administrador y un usuario no registrado.

Trabajaremos de manera colaborativa desarrollando la aplicación por susu funcionalidades. Nos dividiremos ciertas funcionalidades y nos apoyaremos dependiendo de la dificultad. Empezaremos dividiendo el trabajo por archivos html para disminuir las posibilidades de conflictos.Sin embargo, mantendremos el mismo estilo en todos los archivos.
