Pantalla Inicial

En la pantalla inicial de la aplicación web, se encuentra las diferentes opciones de acceso para el usuario (usuario con cuenta existente (login), no registrado (no user), opción para crear su cuenta(sign up)). Adicialmente tendrá un botón para acceder a la pantalla de contacto. (incluir imagen)

Interacción usuario no registrado (Sign up)

Pantalla de Registro

Al elegir la opción de crear una nueva cuenta, el usuario será (redirected) a la sección de registro, Esta sección consiste de un formulario que el usuario tendrá que completar con su información personal: nombre, apellido, edad, etc. Adicionalmente, el usuario completará el proceso de crear su cuenta al dar un nombre de usuario y una contraseña. (incluir imagen)

Interacción usuario registrado

Inicio de sesión

Usuarios que ya tengan una cuenta registrada en la aplicación (login), serán (redirected) a está sección, donde se encuentra la opción de iniciar sesión al ingresar el nombre de usuario y contraseña (incluir imagen)
